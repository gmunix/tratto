# Tratto Frontend

Frontend base do projeto Tratto, criado com React + Vite para a atividade de configuracao base.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run lint:fix`
- `npm run format`
- `npm run test`

## Estrutura

- `src/assets`
- `src/components`
- `src/config`

- `src/pages`
- `src/routes`
- `src/services`
- `src/store`
- `src/styles`
