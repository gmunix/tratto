# Tratto Backend

Backend base do projeto Tratto, criado com Node.js + Express para a atividade de configuracao base.

## Scripts

- `npm run dev`
- `npm run start`
- `npm run lint`
- `npm run lint:fix`
- `npm run format`
- `npm run test`

## Estrutura

- `src/config`
- `src/controllers`
- `src/middlewares`
- `src/models`
- `src/routes`
- `src/services`
- `src/utils`

## Rota inicial

- `GET /api/health`
